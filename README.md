# QKan-plugin-HE7

Für den Datenaustausch mit HYSTEM-EXTRAN 7.8 und 7.9 wurden drei Module als eigenes Plugin zusammengefasst. Die Plugins benötigen (im Gegensatz zu QKan) die zusätzliche Installation eines Firebird-Datenbankservers sowie eine entsprechenden Python-Moduls.

Die Module setzen die QKan-Module voraus und sind nicht eigenständig lauffähig.
